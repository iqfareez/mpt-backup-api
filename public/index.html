MPT JAKIM Backup API

Welcome

This API was not meant for public use. It is designed for the Malaysia Prayer Time app if JAKIM's API is unreachable at that moment. However, there is no harm in using them in small/medium applications.

Data health

Prayer times data will be refresh from JAKIM's server at monthly intervals. Fetcher is scheduled to run (typically) at 8.00 am on the first day of the month.

Hence, this backup database is considered valid in the same month as the fetcher last run date.

Usage

GET https://mpt-backup-api.herokuapp.com/solat/SGR01

where SGR01 is the location code. View full list here.
