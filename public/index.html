MPT JAKIM Backup API

Welcome

First of all, if you ever find this page, take note that this API was not meant for public use. This API is used in Malaysia Prayer Time Android app if JAKIM's API is unreachable at that moment.

Download MPT app JAKIM E-solat status

Data health

Prayer times data will be refresh from JAKIM's server on a monthly intervals. Fetcher is scheduled to run (typically) on 8.00 am on the first day of the month.

Hence, this backup database is consider valid on the same month with the fetcher last run.
